# Shunt / árammérő panel - review checklist

## Feltöltött fájlok ellenőrzése

- [ ] Kapcsolási rajz PDF megvan.
- [ ] PCB felső nézet megvan.
- [ ] PCB alsó nézet megvan.
- [ ] 3D nézet megvan.
- [ ] BOM / alkatrészlista megvan, ha elérhető.
- [ ] Gyártási fájlok megvannak, ha elérhetőek.

## Árammérés

- [ ] Shunt értéke dokumentálva.
- [ ] Shunt teljesítményterhelhetősége dokumentálva.
- [ ] Kelvin bekötés ellenőrizve.
- [ ] Árammérő IC pontos típusa dokumentálva.
- [ ] Mérési tartomány dokumentálva.
- [ ] Offset / gain kalibrációs terv megvan.

## I2C / kommunikáció

- [ ] I2C cím dokumentálva.
- [ ] Pullup ellenállások és feszültségszint ellenőrizve.
- [ ] Fő vezérlővel közös földelés / izoláció tisztázva.

## Szoftverhez szükséges adatok

- [ ] IC típusa.
- [ ] I2C cím.
- [ ] Shunt érték.
- [ ] Max áram.
- [ ] Kalibrációs pontok.
- [ ] Túláram határérték.
