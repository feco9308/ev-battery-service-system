# Relé / terheléskapcsoló rész

## Funkció

Ez a rész kapcsolja a terhelést, illetve a mérési/töltési útvonalakat.

## Tervezett feladatok

- Terhelés relék vezérlése.
- Mérési útvonalak kapcsolása.
- Biztonságos leválasztás hiba esetén.
- Relé vagy kontaktor állapot visszajelzés, ha van.

## Feltöltendő fájlok

```text
schematic.pdf
pcb-top.png
pcb-bottom.png
bom.csv vagy bom.xlsx
gerber.zip
relay-map.md
review-checklist.md
```

## Ellenőrzési fókusz

- Relétekercs meghajtás.
- Flyback dióda / TVS védelem.
- Kontaktus terhelhetőség.
- Bekapcsolási sorrend.
- Reset alapállapot: relé OFF.
- Nyák távolságok nagyáramú és nagyfeszültségű részeken.
- Melegedés és rézszélességek.
