# Csatlakozók és kábelezés

## Funkció

Ez a mappa a panelek közötti csatlakozók, kábelek, pinoutok és harness dokumentáció helye.

## Ide tartozik

- fő vezérlő csatlakozók,
- cellamérő kábelezés,
- balanszer modul csatlakozók,
- CAN kábel és lezárás,
- RS232 kábelezés,
- tápegység csatlakozás,
- nagyáramú vezetékek,
- emergency stop vezetékek.
