# Cella feszültségmérő / isoSPI panel - review checklist

## Feltöltött fájlok ellenőrzése

- [ ] Kapcsolási rajz PDF megvan.
- [ ] PCB felső nézet megvan.
- [ ] PCB alsó nézet megvan.
- [ ] 3D nézet megvan.
- [ ] BOM / alkatrészlista megvan, ha elérhető.
- [ ] Gyártási fájlok megvannak, ha elérhetőek.

## Cellamérés

- [ ] Cellabemenetek sorrendje dokumentálva.
- [ ] Cellacsatlakozó pinout dokumentálva.
- [ ] Bemeneti szűrők értékei ellenőrizve.
- [ ] Cellamérő IC pontos típusa dokumentálva.
- [ ] Open-wire teszt lehetősége tisztázva.

## isoSPI

- [ ] isoSPI transzformátor / csatolás bekötése rendben.
- [ ] Lezárás / polaritás ellenőrizve.
- [ ] Kábel / csatlakozó irány dokumentálva.
- [ ] Izolációs távolságok ellenőrizve.

## Szoftverhez szükséges adatok

- [ ] Cellák száma.
- [ ] Mérő IC típusa.
- [ ] IC lánc sorrendje.
- [ ] Cella indexelés.
- [ ] Hőmérséklet / GPIO csatornák kiosztása, ha vannak.
