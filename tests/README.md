# Tesztek

## Tesztelendő területek

- CAN frame dekódolás.
- CAN parancs encode/decode.
- Heartbeat timeout.
- Emergency stop.
- Biztonsági határértékek.
- Mérési állapotgép.
- Tápegység válasz timeout.
- Balanszer UART timeout.

## Első cél

Unit tesztek a Linux gateway CAN protokoll dekóderéhez.
