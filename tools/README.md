# Fejlesztői eszközök

Ide kerülnek a segédprogramok.

## Tervezett eszközök

- CAN szimulátor.
- Protokoll tesztelő.
- Log parser.
- CSV export ellenőrző.
- Firmware uploader későbbre.

## Első cél

Egy egyszerű CAN szimulátor, amely Linuxon vcan0 interfészen tud teszt státusz frame-eket küldeni a gateway appnak.
